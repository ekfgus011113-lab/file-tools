# 파일딱

사진을 원하는 파일 용량 이하로 줄이는 한국어 웹 도구입니다.

## 첫 MVP 기능

- JPG, PNG, WEBP 사진 선택
- 100KB, 300KB, 500KB, 1MB 또는 직접 목표 용량 입력
- 화질과 해상도를 자동 조절해 목표 용량 이하로 압축
- 압축 전후 용량과 감소율 표시
- 결과 사진 저장
- 모바일 화면 지원
- 서버 전송 없이 브라우저 안에서만 이미지 처리
- 투명 PNG는 형식을 유지하고, 일반 PNG 스크린샷은 JPG로 바꿔 해상도 보존

## 실행

Node.js 22.13 이상과 pnpm이 필요합니다.

```bash
pnpm install
pnpm dev
```

검증:

```bash
pnpm test
pnpm lint
```

## 개인정보

선택한 사진은 서버로 업로드하거나 저장하지 않습니다. 이미지 압축은 사용자의 브라우저 안에서 처리됩니다.
